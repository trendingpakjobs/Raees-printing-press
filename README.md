# Raees Printing Press Website

Professional website for **Raees Printing Press**, located in **Wah Cantt / Taxila**. This project includes a fully responsive 4-page website with interactive elements, premium animations, and user-friendly design.

## Pages
1. **Home** – Hero section with parallax effect, stats, services, why choose us, featured products, CTA, and footer.  
2. **Products** – Showcase printing products with animated cards.  
3. **Portfolio** – Showcase previous projects with animated cards.  
4. **Disclaimer** – SEO-optimized legal information page with blue headings.

## Features
- Responsive design for **desktop and mobile** ✅  
- Hero **parallax + fade animations** ✅  
- Stats, Services, Products, Portfolio **card animations** ✅  
- **Floating WhatsApp button** on all pages ✅  
- Hamburger navigation with smooth animation ✅  
- Footer includes **map, email, WhatsApp, and Instagram icon** ✅  
- Color scheme: **White background, Red headings, Black text** ✅  
- SEO-optimized meta titles, descriptions, and headings ✅  
- Premium **hover effects & microinteractions** for cards ✅
